jQuery Combinate

A simple jQuery plugin for combining or separating text values in separate html form fields.

http://github.com/ftsanjuan/jquery-combinate

$(selector).combinate(options)

Simple Settings

Clicking Combinate! will combine values in the fields with class='combinate' and will insert them combined string into the element with a css selector of '#combinated'

Code:

  $('.form-simple').combinate({
    resultElement: '#combinated'
  });

Additional Settings

Clicking Combinate! will combine values in the fields with class='combinate' except for the third field which has a class of 'exclude-me'

Also, a separator will be added to separate each piece and the final result string will be inserted into #combinated-advanced

Code:

  $('.form-advanced').combinate({
    resultElement: '#combinated-advanced',
    separator: '-',
    exclude: '.exclude-me'
  });

$(selector).decombinate(options)

Simple Settings

Clicking Decombinate! will 'de-combine' (read: separate) the value in the targeted field into pieces (1 character each) and insert them into separate input fields with class='decombinate'.

Note: Because decombinate defaults to 1 character per field, any remaining unused characters are discarded.

Code:

  $('#field-decombinate').decombinate();

Decombinate!

Advanced Settings

Clicking Decombinate! will 'de-combine' (read: separate) the value in the targeted field into pieces (3 characters each) and insert them into separate input fields with class='decombinate-advanced' excluding the field with class='exclude-me-too'

Note: Decombinate will use either the specified pieceLength, the field's maxlength attribute or the default piecelength (1) to determine how many characters are to be inserted per field. Character blocks are inserted into each field until all characters have been used.

Code:

  $('#field-decombinate-advanced').decombinate({
    resultClass: 'decombinate-advanced',
    pieceLength: 5,
    exclude: '.exclude-me-too'
  });
